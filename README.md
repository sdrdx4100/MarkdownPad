# MarkdownPad

マークダウン対応メモ帳アプリケーション

## 概要

MarkdownPadは、C#/WPFで開発されたマークダウン対応のメモ帳アプリケーションです。リアルタイムプレビュー機能、スクリーンショット貼り付け機能など、多彩な機能を備えています。

## 機能

### マークダウン機能
- ✅ マークダウン記法のサポート
- ✅ リアルタイムプレビュー（エディターとプレビューの分割表示）
- ✅ マークダウン書式の挿入ツールバー
- ✅ 見出し、太字、斜体、リスト、コードブロック等のサポート
- ✅ テーブル、タスクリストのサポート

### 基本的なメモ帳機能
- ✅ 新規作成 (Ctrl+N)
- ✅ 開く (Ctrl+O)
- ✅ 保存 (Ctrl+S)
- ✅ 名前を付けて保存 (Ctrl+Shift+S)
- ✅ 元に戻す (Ctrl+Z)
- ✅ やり直し (Ctrl+Y)
- ✅ 切り取り (Ctrl+X)
- ✅ コピー (Ctrl+C)
- ✅ 貼り付け (Ctrl+V)
- ✅ すべて選択 (Ctrl+A)
- ✅ 検索 (Ctrl+F)
- ✅ 置換 (Ctrl+H)

### 画像機能
- ✅ スクリーンショットの貼り付け (Ctrl+V)
- ✅ 画像ファイルの挿入
- ✅ クリップボードからの画像貼り付け
- ✅ 画像の自動保存とマークダウン参照の挿入

### その他の機能
- ✅ 文字数カウント
- ✅ 行番号・列番号表示
- ✅ ワードラップ切り替え
- ✅ プレビュー表示/非表示切り替え
- ✅ 未保存の変更の確認ダイアログ

## スクリーンショット

アプリケーションは以下のようなインターフェースを持っています：

```
┌──────────────────────────────────────────────────────────────┐
│ ファイル  編集  表示  挿入  ヘルプ                              │
├──────────────────────────────────────────────────────────────┤
│ 📄 📂 💾 | ↩️ ↪️ | ✂️ 📋 📎 | B I </> | 🖼️ 🔗 | 👁️           │
├───────────────────────────┬──────────────────────────────────┤
│        エディター          │          プレビュー              │
│                           │                                  │
│  # 見出し                  │  見出し                          │
│                           │  ────────                        │
│  **太字** *斜体*           │  太字 斜体                       │
│                           │                                  │
│  - リスト1                 │  • リスト1                       │
│  - リスト2                 │  • リスト2                       │
│                           │                                  │
└───────────────────────────┴──────────────────────────────────┘
│ 準備完了 | 行: 1 列: 1 | 文字数: 0 | UTF-8                     │
└──────────────────────────────────────────────────────────────┘
```

## 必要要件

- Windows 10/11
- .NET 8.0 Runtime
- WebView2 Runtime (Microsoft Edge WebView2)

## ビルド方法

```bash
cd MarkdownPad
dotnet restore
dotnet build
```

## 実行方法

```bash
dotnet run --project MarkdownPad
```

または、ビルド後に生成される実行ファイルを直接実行してください：

```bash
./MarkdownPad/bin/Debug/net8.0-windows/MarkdownPad.exe
```
